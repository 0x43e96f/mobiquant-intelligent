# Layer 1: 数据层 (Data)

> 让 AI 能看到市场

AI 再聪明，没有数据就是瞎子。这一层解决的问题是：怎么让 AI 获取市场信息。

## 要解决的问题

- 行情数据从哪来？（交易所 API、聚合平台）
- 怎么处理多个数据源？（一个不够用，多个怎么聚合）
- 数据缓存（不能每次都重新请求，太慢也容易被限流）
- 数据质量（不同来源的数据可能冲突，以谁为准）
