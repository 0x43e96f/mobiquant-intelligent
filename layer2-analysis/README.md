# Layer 2: 分析层 (Analysis)

> 让 AI 能思考

数据有了，下一步是让 AI 学会分析。不是罗列数据，是建立判断框架。

## 要解决的问题

- 怎么让 AI 分析一只标的？（不是列数字，是找核心矛盾）
- 怎么防止 AI 顺着你说话？（反确认偏误）
- 怎么让两个 AI 互相辩论？（多空辩论机制）
- 怎么评估风险？（不只看收益，先看会亏多少）
